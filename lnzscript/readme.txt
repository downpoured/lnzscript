Powerful JavaScript libraries for Win32 automation
by Ben Fisher
=================

To Get Started:
Run the Installer.
Open the "Lnz Editor" from the Start menu. This is a code editor specifically for the Launchorz project. 
Begin typing a script, such as Mouse.move(1,1); alert('hello');. 
Choose run script from the Run menu (or press F5) to run. 

(Scripts can also be run from the command line using lnzscript.exe).
