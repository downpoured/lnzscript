Easily automate Windows with JavaScript.

=================

To Get Started:
Download the latest version from
https://github.com/moltenjs/lnzscript
and run the installer.

Open the "Lnz Editor" from the Start menu. This is a code editor specifically for Lnzscript.
Type a script, such as Mouse.move(1,1); print('hello');. 
Choose run script from the Run menu (or press F5) to run. 

You can also run a scripts from the command line, with lnzscript.exe.
